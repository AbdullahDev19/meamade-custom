# restore - Stock Location Next Module Reference

This documentation provides a reference to the `restore` method. This belongs to the Stock Location Next Module.

This method is used to restore a stock location or multiple stock locations that were previously deleted using the [softDelete](stock_location_next.IStockLocationServiceNext.softDelete.mdx) method.
